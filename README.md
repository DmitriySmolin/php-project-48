## About

A difference calculator is a program that determines the difference between two data structures

## Requirements

* php 7.1+
* composer 2.2.6
* make 4.3

## Setup

```sh
$ git clone https://github.com/DmitriySmolin/php-project-48.git

$ cd php-project-48

$ make install
```
